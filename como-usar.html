Como usar o aplicativo de Realidade Aumentada

Para aproveitar ao máximo o nosso aplicativo, siga os passos abaixo:

1. Acesse a experiência de Realidade Aumentada clicando no botão “Iniciar Experiência de RA” na página inicial.
2. Permita o acesso à câmera do seu dispositivo para que o aplicativo possa reconhecer os marcadores nos livros didáticos.
3. Aponte a câmera para as páginas indicadas do livro Touchstone para visualizar os objetos virtuais em 3D.
4. Interaja com os conteúdos digitais para reforçar o aprendizado do vocabulário e das estruturas em inglês.
5. Caso tenha dúvidas, retorne à seção “Como usar” ou entre em contato conosco.

O aplicativo foi desenvolvido para funcionar em dispositivos móveis e computadores modernos, garantindo acesso fácil para todos os alunos do CELIFF.
